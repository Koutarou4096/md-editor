# MARKDOWN EDITOR

これは元々僕がMDエディターが欲しくなって、どれも導入がめんどくさいし、なんなら探すのもめんどくなったので自分で作ったやつです。

僕には完全に自作できるほどの技術力と時間がないので、一部バグ修正などを除いてほとんどClaudeに作ってもらいました。

バグ修正済みとはいったものの、まだ見つけられてないバグもあるかもしれないので、あったらごめんなさい。そのうち問い合わせフォームを作ろうと思います（覚えてたら）。
