## はじめに
これは元々自分用に作ったものですが、便利だし他の人にも使ってほしいと思い公開しました。

僕には完全に自作できるほどの技術力と時間がないので、一部デザイン修正やバグ修正などを除いてほとんどClaudeに作ってもらいました。

## 機能・できること
マークダウン方式を用いることにより、比較的簡単に見栄えの良い文書を作ることができます。

作成した文書を画面に表示する用途だけでなく、.mdやHTML形式での保存、PDF化、印刷も可能となっています。<br>※ 一部 β 版機能も含みます。

編集内容を一時的にブラウザ内に保存することにより、間違えてページを再読み込みしたり、タブを消してしまったりしても、もう一度開いたときに編集内容が復元されるようになっています。<br>※ 一部のブラウザでは正常に動作しない場合があります。

保存機能はあくまでブラウザ内に一時的に保存するものであり、編集内容が漏洩したり、監視されたりできない仕様となっているので、安心してご利用ください。
