僕には完全に自作できるほどの技術力と時間がないので、一部バグ修正などを除いてほとんどClaude先生に作ってもらいました。

バグ修正済みではありますが、まだ見つけられてないバグもあるかもしれないので、あったらごめんなさい。そのうち問い合わせフォームを作ろうと思います（覚えてたら）。
